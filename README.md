# SistemaCMMF - Harmony Hub

CRM e sistema de gestão do Centro de Música Murilo Finger.

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Supabase (PostgreSQL)
- Recharts
- Lucide Icons
- React Router

## Setup

```bash
npm install
cp .env.example .env  # Configure as variáveis
npm run dev
```

## Páginas

- **Dashboard** — KPIs, gráficos, contatos recentes
- **Contatos** — CRUD de leads com filtros
- **Aulas Experimentais** — Gestão de aulas agendadas
- **Matrículas** — Registro de matrículas e faturamento
- **Follow-up** — Acompanhamento de retornos
- **Relatórios** — Funil de conversão e análises
- **Configurações** — Professores, Cursos, Salas, Planos
